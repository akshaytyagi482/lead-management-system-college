# Week 2 - Interactive JS Elements

## Focus: DOM Manipulation & Events

### Deliverable
Interactive elements with event handling added to the Lead Management System.

### Features Implemented
- **Theme Toggle**: Dark mode / Light mode switch with smooth CSS transitions
- **Dynamic Content Loading**: Leads table is populated dynamically via JavaScript
- **Animated Counters**: Dashboard stats animate from 0 to the actual value on page load
- **Dynamic Reports**: Source and status reports are generated from data using DOM manipulation
- **Event Handling**: Click events for theme toggle, dynamic DOM element creation

### How to Run
Open `index.html` in any modern web browser.

### Technologies Used
- HTML5
- CSS3 (Transitions, Dark Mode)
- JavaScript (DOM Manipulation, Events, Dynamic Rendering)
