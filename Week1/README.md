# Week 1 - Static Web Structure

## Focus: HTML5/CSS Responsive Layout

### Deliverable
A responsive static page for the **Lead Management System** with mobile-first design.

### Features Implemented
- Responsive HTML5 layout with semantic elements
- CSS Grid for dashboard stats and reports sections
- Responsive navigation bar
- Leads table with styled badges
- Mobile-first media queries (768px and 480px breakpoints)
- Clean, modern design with hover effects and transitions

### How to Run
Simply open `index.html` in any modern web browser.

### Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
